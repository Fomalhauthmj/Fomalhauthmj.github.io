Codeforces Global Round 1

发表于 2019-02-13 更新于 2020-02-24 分类于 Codeforces

做题笔记之Codeforces Global Round 1

比赛链接：Codeforces

题解链接：Codeforces

水平有限，只做前三题……

A - Parity

我做的时候看了下数据规模，蛮大的直接实现的话应该要超long long，然后就开始分析规律。
联系下二进制n&1判断奇数应该可以看出来。
对于奇数的进制b^k仍为奇数，统计奇数位是不是偶数即可。

B - Tape

贪心考虑，需要将线段长度短的连起来，长的单点处理。
解法一是我做的排序后取较短模拟处理，略显麻烦。
参考题解可以用一种更简单的逆向做法，解法二。

C - Meaningless Operations

一开始做的时候打表容易发现存在规律，难处理的是num=2^x-1的情况，可以预处理跑一遍然后直接查表即可。
以下为英文题解：

Denote the highest bit of a as x (that is largest
number x, such that 2^x≤a) and consider b=(2^x−1)⊕a.
It’s easy to see that if a≠2^x−1 then 0<b<a holds. In
this, gcd is maximum possible, because a&b=0 и gcd
(2^x−1,0)=2^x−1.
当a不等于2^x-1时 如a=1001 则当b=0110时取max_gcd 此时gcd(a^b,a&b)=gcd(2^x-1,0)=2^x-1

Now consider the case when a=2^x−1. This implies gcd
(a⊕b,a&b)=gcd(2^x−1−b,b)=gcd(2^x−1,b), since gcd(x,x
+y)=gcd(x,y) (for all x and y), hence it’s sufficient
to find the largest non trivial divisor of a — it
will be the desired answer.

Finding the largest divisor can be done in sqrt
time which leads to the time O(q√m) or it’s possible
to calculate answers for all a=2^x−1 beforehand.
这里推荐用预处理，我参考题解用sqrt(a)寻找最大因子时超时
了。也可能是方法不正确。
